Testes

Obs. Para garantir os teste, degite o comando "clear" em qualquer um dos
servidores de realizar os teste, pois o resultado vai ser influenciado caso ja tenha algo no banco. 

Teste Ok e NOK
Descricao
 - Vao ser inseridos valores para testar e as repostas do servidor
 - O OK para testar sucesso e NOK falha

Como executar
 - Os tres servidores tem que estar online
 - Entre na pasta do projeto pelo terminal (\etapa3)
 - Execute o script "test-oknok.sh"

Teste OPERACOES
Descricao
 - Vai ser feitas todas as operacoes em 15 chaves (2 a 17)
 - Primeiro um DELETE e depois um CREATE
 - Por fim um UPDATE
 - Vao ser inseridos 5 chaves em cada servidor
 - No final o banco deve ter os valores de A a O

Como executar
 - Os tres servidores tem que estar online
 - Entre na pasta do projeto pelo terminal (\etapa3)
 - Execute os script "test-operacoes.sh"

Teste Consistencia
Obs. Nao conseguimos reiniciar os servidores de forma automatizada,
entao e necessario reinicia-los manualmente
Descricao
 - Teste para testar a consistencia do banco

Como executar
 - Faca o "Teste OPERACOES"
 - Feche os 3 servidores, digitando close nos respectivos terminais
 - Inicie os 3 servidores
 - Faca o "Test READ"

Teste Ordem de Excecucao
Descricao
 - Teste que vai inserir 1000 valores, no qual o atual vai ser a soma
do anterior + 1
 - O primeiro valor vai ser 0 com a chave 1000
 - O ultimo vai ter o valor de 1001 com a chave 2000
 - A chave 2000 vai ser lida e printada na tela

Como executar
 - Os tres servidores tem que estar online
 - Entre na pasta do projeto pelo terminal (\etapa3)
 - Execute os script "test-ordem-exec.sh"
